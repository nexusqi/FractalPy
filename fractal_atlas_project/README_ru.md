# Fractal V1 / Fractal Atlas (черновой прототип)

Этот архив содержит простой прототип фрактального декодера V1 и заготовку под «Фрактальный Атлас».

Идея:

- Вместо того, чтобы подавать в сеть картинку или длинный текст целиком, мы подаём **координату** (адрес) `x`.
- Через **Fourier Features** (спектральное расширение) координата превращается в богатый по частотам вектор признаков.
- Этот вектор проходит через **фрактальное ядро** (итерационный слой), которое многократно применяет одни и те же веса.
- На выходе мы получаем распределение по символам (или другим токенам).

Варианты использования:

1. `train_single_book.py` — пример оверфита одной строки текста:
   - координата `x = i / N` → целевой символ `text[i]`.
   - модель учится «зазубривать» соответствие «адрес → буква».

2. `atlas_keys.py` — заготовка под «Фрактальный Атлас»:
   - общий фрактальный декодер (мастер-фрактал),
   - для каждой «книги» или «знания» свой латентный ключ,
   - веса фрактала можно заморозить и обучать только ключ.

Файлы в архиве:

- `fractal_decoder.py` — реализация FractalDecoder V1 (PyTorch).
- `train_single_book.py` — простой цикл обучения для одной «книги».
- `atlas_keys.py` — пример того, как привязать латентный ключ к декодеру.
- `data/example_text.txt` — небольшой текст для тестов.
- `requirements.txt` — минимальный список зависимостей (PyTorch + numpy).

Все комментарии внутри кода — на русском, чтобы было удобно делиться и обсуждать.
